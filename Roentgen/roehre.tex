\subsection{Röntgenröhre}
Für die Erzeugung von Röntgenstrahlung kann eine Röntgenröhre verwendet. Eine schematische Darstellung einer Röntgenröhre ist ein Abb. \ref{fig:roehre} zu sehen.
Die Röntgenröhre besteht Hauptsächlich aus einer Glühwendel, einer Anode, und einer Vakuumglashülle. Die Glühwendel wird zum emittieren von Elektronen verwendet. Mit einer Potentialdifferenz von 10 bis 100 kV werden die Elektronen zur Anode hin beschleunigt.
Die beschleunigten Elektronen treffen, auf die Anode, wo sie durch Stöße Röntgenstrahlen emittieren. Jedoch wird nur zu 1\% Röntgenstrahlung erzeugt, der Rest geht in Wärme über, wodurch es nötig wird die Anode zu kühlen.
