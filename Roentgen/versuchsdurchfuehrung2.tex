\subsection{Versuchsdurchführung}
Nun soll mit dem Diffraktometer die Zusammensetzung einer unbekannten Pulverprobe bestimmt werden. Mit dem Diffraktometer werden dazu Diffraktogramme erstellt, aus denen mittels einer Datenbank qualitativ die Probenzusammensetzung ermittelt werden kann. Ebenfalls sollen unabhängig von der Auswertung mithilfe der Datenbank einige Netzebenenabstände d aus den Diffraktogrammen bestimmt werden.
