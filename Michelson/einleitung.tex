\section{Einleitung}
%einleitung zu dem experiment.
%auf die einstellungen, die vor dem versuch gemacht werden, eingehen, oder auf eine anleitung dazu verweisen.
%---------------------------------------------------------------------------------------------
%hinter der einleitung kann der allgemeine theoretische hintergrund in einer zusätzlichen section erklärt werden
Mit einem Michelson Interferometer soll das Infrarotstrahlungsspektrum eines He-Ne-Lasers und eines Muffelofens, welcher als Schwarzkörperstrahler verwendet wird, untersucht werden. Mithilfe von Interferenzfiltern kann das Strahlungsspektrum des Muffelofens eingegrenzt werden. Das Spektrum wird in einem Bereich von (\SI{3,31}{$\mu$m}) dicht neben der Wellenlänge des He-Ne-Lasers (\SI{3,39}{$\mu$m}) eingeschränkt. Durch Superposition beider Strahlungsquellen entsteht das Interferenzmuster einer Schwebung, welches in diesem Versuch ebenfalls analysiert werden soll.
